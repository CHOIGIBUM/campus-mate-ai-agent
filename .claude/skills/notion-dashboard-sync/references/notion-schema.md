|---|---|
| 공모전명 | 제목 |
| 원문 링크 | 원문 URL |
| 출처 | 수집 소스 |
| 유형 | 공고 유형 |
| 포스터 | 이미지 |
| 마감일 | 접수 마감 |
| 행사일 | 행사·시작일 |
| 참가 자격 | 대상 조건 |
