# 필드 출처 우선순위

| 출처 | 기본 신뢰도 | 사용 기준 |
|---|---:|---|
| JSON-LD | 0.90–1.00 | 명시된 구조화 데이터 |
| Next.js 상태 | 0.90–1.00 | 애플리케이션 상태 데이터 |
| 화면 HTML | 0.75–0.95 | 화면 라벨·본문 |
| OCR | 0.45–0.80 | 이미지·렌더링 텍스트 |
| Vision | 0.45–0.80 | 포스터 시각 정보 |
| 수동 입력 | 별도 지정 | 출처를 명시한 입력 |

높은 우선순위 출처와 충돌하는 낮은 우선순위 값은 자동 선택하지 않습니다. 해결되지 않은 핵심 충돌은 `NeedsReview`로 처리합니다.
