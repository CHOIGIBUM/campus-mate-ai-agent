# Notion Schema

Core properties may include:

- 공모전명 / Title
- 원문 링크
- 출처
- 유형
- 포스터
- 마감일
- 행사일
- 참가 자격
- 제출물
- 시상/혜택
- 적합도
- 우선순위
- 추천 이유
- 상태 (legacy alias `상태(새)` supported)
- 일정 충돌
- 요약 (legacy alias `메모` supported)
- stable opportunity ID
- calendar event IDs
- sync error

The implementation only adds missing required properties and preserves existing pages.
